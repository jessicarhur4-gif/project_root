🧾 과제 수행 기록
이번 과제에서 다음 Git 명령어를 사용하였다.

- `git init` : 로컬 저장소 초기화  
- `git add .` : 모든 파일 스테이징  
- `git commit` : 변경 사항 커밋  
- `git push` : 원격 저장소(GitHub)로 업로드  
- `git pull` : 원격 저장소의 최신 버전 가져오기  
- `git log` : 커밋 기록 확인  
- `git status` : 현재 작업 트리 상태 확인
